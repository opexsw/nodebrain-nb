Issuing NodeBrain Commands

    http[s]://hostname:port/:request

To select the page for entering commands to Webster's host NodeBrain, the request is "nb".   You can view the command entry page in a new window or tab using the link below.

    nb
To issue a command bypassing the command entry page, the request looks like this.

    nb?command

Try this list of commands by selecting them one at a time in sequence.   Use your browser's back button to return to this page after each selection.

  • -ls -al

  • define rDoLs on(a=1 and b=2):-ls -al

  • assert a=1,b=7;

  • show -t

  • assert b=2;

If you selected the links in sequence, the result of the last command link should have looked a lot like the first.   If you understand the second command, you know why.   If not, you should return to the NodeBrain Tutorial document and review the tutorials up to and including the Rules tutorial.

In the window or tab opened by the first link on this page, nb, take some time to experiment by entering more NodeBrain commands.
