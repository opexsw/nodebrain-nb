Browsing Files

    http[s]://hostname:port/:request

A request of "file" will take you to a page that enables navigation of the file system and text file browsing. You can view this page in a new window or tab using the link below.

    file
By default, navigation starts at the working directory of the NodeBrain process hosting Webster.   To go directly to a different directory or file, the request looks like this.

    file?path

Try this list of commands.   Use your browser's back button to return to this page after each selection.

  • webster1

  • webster2

  • webster.log

  • /etc/init.d

In the window or tab opened by the first link on this page, file, take some time to experiment by navigating the file system and browsing files.
